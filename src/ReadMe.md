# 💰 Poupyg! - Fintech

O **Poupyg** é uma aplicação voltada para pessoas que buscam organizar sua vida financeira, oferecendo controle total sobre receitas, despesas e investimentos em uma interface intuitiva.



---

## 📋 Funcionalidades Principais

* **Dashboard (Home):** Painel central com todos os dados importantes para o monitoramento imediato das finanças.
* **Movimentações:** Gestão completa de entradas (receitas) e saídas (despesas). Permite visualizar o extrato, editar descrições e consultar o saldo total calculado automaticamente pelo sistema.
* **Investimentos:** Área dedicada para acompanhar o patrimônio investido, com suporte para exclusão e edição de ativos.
* **Perfil:** Central de dados do usuário, contendo informações cadastrais e opção segura para logout.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** ReactJS, Vite
* **Backend:** Java, Spring Boot
* **Banco de Dados:** Oracle SQL
* **Gerenciadores:** Maven (Backend) e NPM (Frontend)

## 🚀 Instruções de Inicialização

Siga os passos abaixo para rodar o projeto localmente:

### 1. Backend (Java/Spring Boot)
1. Abra o seu projeto Java em uma IDE (como IntelliJ ou Eclipse).
2. Localize a classe principal da aplicação e execute o método `run`.
3. O servidor backend será inicializado, conectando-se ao banco de dados.

### 2. Frontend (ReactJS)
1. Abra o terminal na pasta do projeto frontend.
2. Execute o comando para iniciar o servidor de desenvolvimento:
   ```bash
   npm run dev
3. Siga o link localhost gerado.

4. ## Atenção : certifique-se de ter o node instalado em sua máquina ##
